# 🌐 Personal Portfolio Website

## 📌 Description
This is a personal portfolio website built using HTML, CSS, JavaScript, and Bootstrap.  
It showcases my profile, skills, projects, and resume in a clean and responsive design.

## 🚀 Features
- Responsive design (works on mobile & desktop)
- Smooth navigation
- Personal profile section
- Projects showcase
- Resume download option
- Clean and modern UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap
